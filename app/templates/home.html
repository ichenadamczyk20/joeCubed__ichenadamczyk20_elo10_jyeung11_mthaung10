Search for images of things. Each thing should be separated by a comma. If a thing exists as a wikipedia page, you will have an image for it returned.

Some examples:
pie, a, dream, apple pie, pi
wolf, fat, car, tar, water
gudetama, anime, science, death, bug

Notes:
1) This may also return gifs.
2) Each query will always return the same image or gif.
3) Have fun with it!
